# HecateVM
Tiny register-based VM written in C.

## Instructions
| code | name     |
|:----:|:---------|
| 0x0  | SETR_IMM |
| 0x1  | SETC_IMM |
| 0x2  | SETT     |
| 0x3  | TSET     |
| 0x4  | TSETL    |
| 0x5  | TSETE    |
| 0x6  | TSETG    |
| 0x7  | TSET_IMM |
| 0x8  | TCMP     |
| 0x9  | TADD     |
| 0xA  | TSUB     |
| 0xB  | TMUL     |
| 0xC  | TDIV     |
| 0xD  | TGET     |
| 0xE  | TPUT     |
| 0xF  |          |
